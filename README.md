## Welcome to my profile page.
My name is Frank, and I'm a researcher and passionate educator currently working in the higher education sector = and more importantly in the GitHub context, I'm also a hopeless technology enthusiast.

As my work focuses on project management and my main hobby is home automation, GitHub is an interesting area for me: I get to apply my work knowledge to my hobby.
So, this page here *mostly* focuses on my hobby.

Note that I have just started working with GitHub more extensively in 2026. Before that, I never used it in a structured way. That said, I'm looking forward to putting my projects on here over time - and who knows, maybe someone in the community finds some it helpful or at least interesting.  
So, all of this here, including the page you're reading right now, is a work in progress.
